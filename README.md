# kao-cli

a cli used to install react-n-page project


## install

```
$ npm install kao-cli -g
```

## create 

```
$ kao init myproject
```

## dev&build

```
$ cd myproject

$ npm install

$ npm run dev //development

$ npm run build // production
```

## start

```
$ npm start
```
